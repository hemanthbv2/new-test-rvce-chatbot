# RVCE Chatbot AI 🚀

An advanced, conversational AI chatbot designed for **RV College of Engineering (RVCE)**. This chatbot provides students and parents with instant access to department information, HOD details, admission circulars, placement statistics, and campus life updates.

## ✨ Features

- **Comprehensive Knowledge Base:** Integrated data for all 16 UG and 14 PG departments.
- **Dynamic Response Engine:** Automatically renders rich UI cards for departments with links to Syllabus, Labs, and Faculty.
- **Advanced Intent Matching:** Supports common college abbreviations (e.g., `prgm`, `dept`, `plcmnt`) and fuzzy matching.
- **Leadership Directory:** Verified Head of Department (HOD) and Dean information.
- **Voice Enabled:** Supports text-to-speech feedback for a more interactive experience.
- **Campus Life & Facilities:** Quick access to info on Hostels, Sports, NCC, NSS, and Innovation Teams.
- **WordPress Ready:** Designed as a plug-and-play asset for the official RVCE WordPress ecosystem.

## 🛠️ Technology Stack

- **Core:** JavaScript (ES6+), HTML5, CSS3
- **Scraping:** Node.js, Puppeteer (for data extraction)
- **Deployment:** Vercel / WordPress Plugin

## 📂 Project Structure

- `/rvce-chatbot`: Main WordPress plugin directory.
- `script.js`: Core chatbot logic and knowledge base.
- `style.css`: Premium UI styling with glassmorphism effects.
- `scrape_test.js`: Utility script for departmental data scraping.

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` to test the chatbot locally.
3. To deploy on WordPress, zip the `rvce-chatbot` folder and upload as a plugin.
